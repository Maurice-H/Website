## ADDED Requirements

### Requirement: Back-to-Top Button
A button should allow users to easily return to the top of the page.

#### Scenario: Scrolling down the page
- **WHEN** the user scrolls down past a predefined threshold (e.g., 300px)
- **THEN** the Back-to-Top button fades into view.

#### Scenario: Clicking the Back-to-Top button
- **WHEN** the user clicks the Back-to-Top button
- **THEN** the window scrolls smoothly to the top of the page.
