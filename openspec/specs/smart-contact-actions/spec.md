# Capability: Smart Contact Actions

## Purpose
Streamlines communication by providing direct actions (link, copy) for different contact channels.

## Requirements

### Requirement: Functional Actions
The system SHALL support two primary actions for contact methods:
1. **Link**: Opens an external URL in a new tab.
2. **Copy**: Copies a value to the clipboard and provides visual feedback (toast).

### Requirement: Toast Notifications
The system SHALL provide a lightweight notification system to confirm user actions (e.g., "Copied to clipboard").
