# Capability: i18n Support

## Purpose
Enables multi-language support (German/English) for the portfolio, catering to a diverse professional audience.

## Requirements

### Requirement: Multi-language Catalogs
The system SHALL support `de` and `en` locales via JSON message catalogs.

### Requirement: Locale Switcher
The system SHALL provide a UI toggle to switch between supported languages.

### Requirement: Persistence
The chosen locale SHALL be reactive and affect the entire UI, including metadata and document language attributes.
