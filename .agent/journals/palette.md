## 2026-04-30 - Enhanced Form Accessibility and Focus States
Learning: Custom form inputs within a glassmorphism/themed UI (like ContactForm.vue) often rely purely on placeholder text and lack proper semantic `<label>` associations for screen readers. Furthermore, the focus states on interactive elements were not visually distinctive enough for keyboard navigators, especially on the "Send Transmission" submit button which lacked `focus-visible` ring styling entirely.
Action: Implemented explicit, screen-reader-only `<label>` elements (`sr-only`) linked via `id` attributes to each input and textarea. Added `required` attributes. Enhanced both inputs and the submit button with `focus-visible` Tailwind classes (`focus-visible:ring-1`, `focus-visible:ring-finished-accent`, etc.) to provide clear, accessible keyboard focus indicators that align with the cosmic/sci-fi aesthetic.
